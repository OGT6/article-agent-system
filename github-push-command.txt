# GitHub Push Command with Personal Access Token

# Replace YOUR_TOKEN with the token you just copied (starts with ghp_)
git push https://YOUR_TOKEN@github.com/OGT6/article-agent-system.git main

# Example (DO NOT use this fake token):
# git push https://ghp_1234567890abcdef@github.com/OGT6/article-agent-system.git main

# After successful push, set upstream:
git branch --set-upstream-to=origin/main main

# For future pushes, you can just use:
# git push
